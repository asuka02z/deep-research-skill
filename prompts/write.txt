## Your Task

Write all pending sections in order. For each section: read its retrieved info, write content, save it, and update state.json before moving to the next.

   - Must include 1-12 citations in `[[textidN]]` or `[[textid1, textid2]]` format

   - Language must match the user query

   d. If this is NOT the last section, keep `state = "write"`
   e. Write state.json

After the LAST section is written, do NOT change the `state` field — leave it as "write". The state transition will be handled externally. Just report back that all sections are complete.
